# My Blog

https://gabisoamn.github.io

## New Post

    $ cd blog
    $ hugo new posts/[title].md

## Build Blog

To run it locally:

    $ git submodule init
    $ git submodule update
    $ hugo server -D

Build static pages:

    $ hugo -F
